# RAG-Chatbot-ELIJAH
A chatbot that reads documents and answers questions, built as part of my journey into AI engineering and integration.
Overview: ELIJAH is a RAG chatbot that answers questions based on uploaded CVs/documents.

Features: Upload, extract, chunk, embed, store, query.

Setup: pip install -r requirements.txt (include streamlit, langchain-openai, faiss, pdfplumber, python-docx).

Usage: Run "streamlit run Elijah.py", upload files, and ask questions.

Roadmap: Chat history, persistent FAISS index, semantic chunking, UI polish.
